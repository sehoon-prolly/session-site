# Workshop Site Files

- `static-site/`: HTML/CSS/JS 정적 버전
- `react-site/`: Vite + React 버전
- `tools/`: React 실행용 로컬 Node 런타임
- `run-static-site.sh`: 정적 버전 실행 스크립트
- `run-react-site.sh`: React 개발 서버 실행 스크립트
- `build-react-site.sh`: React 빌드 스크립트

## Quick Start

정적 버전:

```bash
./run-static-site.sh
```

React 버전:

```bash
./run-react-site.sh
```
